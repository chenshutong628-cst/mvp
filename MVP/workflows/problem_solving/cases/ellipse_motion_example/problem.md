# 椭圆上的动点问题（动态组件测试）

## 题目描述

已知椭圆 C: x²/16 + y²/9 = 1，A 为椭圆的右顶点，点 M、N 在椭圆上运动，且使得 ∠MAN = 60° 恒成立。

（1）求证：线段 MN 的长度为定值；
（2）当点 M、N 运动时，求三角形 AMN 面积的最大值。

## 考察要点

- 椭圆的参数方程
- 动点轨迹问题
- 动态几何可视化
- **新增**：使用 ValueTracker 和 always_redraw 实现点的运动动画

## 动画要求

1. **场景 1**: 绘制椭圆，标注顶点 A
2. **场景 2**: 展示点 M 在椭圆上运动（使用 DynamicPointOnCurve）
3. **场景 3**: 根据角度约束，展示对应的点 N 位置
4. **场景 4**: 动态显示 MN 长度和三角形 AMN 面积
5. **场景 5**: 展示面积最大值时的位置

## 关键技术点

- 使用 `EllipsePoint` 创建椭圆上的动点
- 使用 `always_redraw` 实现动态更新
- 使用 `DynamicLine` 连接动态点
- 使用 `DynamicTriangle` 展示动态三角形
- 使用 `ValueTracker` 控制动画参数
